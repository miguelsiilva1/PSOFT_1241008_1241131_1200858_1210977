# Glossary

**Terms, Expressions and Acronyms (TEA) must be organized alphabetically.**

| **_TEA_**                     | **_Description_**                                                                                                                                                                     |                                       
|:------------------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Network**                   | A term used to characterize a group of routes.                                                                                                                                        |
| **Manufacturer**              | A company or entity that designs, builds, test, sells, and maintains aircraft or aircraft engines.                                                                                    |
| **Route**                     | A term used to specify a flight between two airport.                                                                                                                                  |
| **Maintenance Part**          | An expression used to specify a specific aircraft component that needs maintenance.                                                                                                   |
| **Backoffice Operator**       | A professional or team responsible for managing the administrative and operational support functions related to aircraft operations.                                                  |
| **ATCC**                      | An acronym used to identify a Air Transport Company Collaborator.                                                                                                                     |
| **Maintenance Technician**    | A professional who inspects, repairs, and maintains aircraft to ensure they are safe and operational.                                                                                 |
| **Maintenance Supervisor**    | A professional who supervises a Maintenance Technician.                                                                                                                               |
| **IATA Code**                 | Acronym for International Air Transport Association, and is a unique three-letter identifier assigned to airports and cities.                                                         |
| **Runway**                    | A term used to characterize a track where aircrafts poise and takeoff.                                                                                                                |
| **Template**                  | A structured tool used to ensure proper inspection, maintenance, overhaul or modification of aircraft systems.                                                                        |
| **Airplane Certification**    | A term used to characterize the permission, owned by an aircraft, to fly to/from an airport.                                                                                          |
| **Checklist**                 | An document used to ensure that all tasks are completed to keep the aircraft safe and operational.                                                                                    |
| **Fleet**                     | A term used to characterize a group of aircrafts.                                                                                                                                     |
| **Completion Notes**          | An expression used to describe a group of, optional, observations registered after a maintenance.                                                                                     |
| **Technical Diagram**         | A detailed representation that shows the various component and systems of an aircraft.                                                                                                |
| **Assignment**                | A term used to express the same as "flight".                                                                                                                                          |
| **Graphical Representations** | A diagram and charts that illustrate various aspects of an aircraft".                                                                                                                 |
| **GeoJSON**                   | Is an open standard format for representing simple geographical features and their non-spatial attributes, based on JSON.                                                             |
| **KML**                       | Acronym for Keyhole Markup Language, and is  an XML notation for expressing geographic annotation and visualization within two-dimensional maps and three-dimensional Earth browsers. |
